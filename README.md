# 💳 Credit Risk Prediction System

A complete end-to-end **Machine Learning project** that predicts whether a loan applicant is **likely to default or not**, helping financial institutions reduce credit risk.

This project demonstrates the full **data science workflow**, including data preprocessing, model training, and deployment using a dashboard interface.

---

## 📌 Project Overview

Credit risk prediction is essential for banks and financial institutions to decide whether to approve a loan.

This project includes:

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training using Random Forest
- Model serialization using Pickle
- Interactive dashboard for predictions

The trained model predicts the **credit risk of loan applicants based on financial attributes.**

## 🔧 How to Run the Project

### Clone the repository

```
git clone https://github.com/Devenzz/Credit-Risk-Prediction.git
```

### Navigate to project folder

```
cd Credit-Risk-Prediction
```

### Install dependencies

```
pip install -r requirements.txt
```

### Run the application

```
python app.py
```

or

```
streamlit run dashboard.py
```
